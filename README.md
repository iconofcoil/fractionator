# fractionator
Fractions Calculator

Command line app to calculate basic operations with fraction numbers.

Fraction numbers format is 3_1/7 where

  3 = whole number (optional)
  _ = whole separator (optional)
  1 = numerator
  / = numerator/denominator
  7 = denominator
  
Input examples:

? 1/5 + 1/7
? 3_2/3 - 2/6
